# english-sentence-devider
DUO3.0(復習編)の英文を分割する

## Usage
Mac前提

Juliusをダウンロードしてインストールしておく(adintool用)
https://github.com/julius-speech/julius/releases
```
tar xvzf julius-4.5
cd julius-4.5
./configure
make
sudo make install
```

soxをインストール
```
sudo chown -R $(whoami) /usr/local/lib/pkgconfig
brew install sox
```
